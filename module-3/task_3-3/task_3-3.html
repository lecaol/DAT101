    }

  7. [Explain what happens when we use recursive functions.]

    8. [En rekursiv funksjon kaller seg selv og løser problemer ved å dele dem opp i mindre deler. Den har en "base case" som stopper rekursjonen og en rekursiv case som kaller funksjonen på nytt. Hvert nye kall nærmer seg base case. Hvis ikke man har en base case vil funksjonen kalle seg selv uendelig og føre til en stack overflow.
